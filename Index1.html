QR Code Based Smart Attendance Management System

Automating traditional attendance systems using secure, time-limited QR authentication integrated with structured database architecture.

Problem Statement

Traditional attendance systems face operational and security challenges such as proxy attendance, manual record keeping, and lack of real-time tracking in large classrooms.

Solution Approach

QR-based authentication with time-limited tokens validated against a database.

Innovation

Eliminates proxy attendance while enabling automated Excel export and analytics.

System Architecture

Backend

Flask API with MySQL database.

QR Validation

Time-limited dynamic QR codes.

Analytics

Automated reports & dashboards.

Live Execution

VIEW DEMO

Results & Impact

Attendance marking reduced from over 5 minutes to under 30 seconds, ensuring accuracy and eliminating proxy attendance.
